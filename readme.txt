== Description ==

Show snow falling on your website based on current weather conditions where you live.

= How do you determine whether it is snowing? =

A free API key from [Open Weather Map](https://openweathermap.org), and your coordinates are required for this plugin to get the current weather forecast where you live. The forecast is cached and updated hourly to keep your site running fast.

= What do I do if it's snowing outside but not on my website?!  =

You can always manually set your website to snow, or manually disable the snow. Updating the plugin options will also force a refresh of the weather forecast.
